# Filu Support

Public issue tracker for [Filu](https://filu.co/), a local-first file conversion app.

Filu helps you convert files quickly while keeping supported conversions on your device. Files do not need to be uploaded to a server for local conversions.

Use this repository to:

- Request support for a new file type
- Request a new conversion, such as `MOV to MP4` or `PDF to PNG`
- Suggest product improvements
- Report problems with the website or app
- Ask for clarification about file support

This repository does not contain the Filu source code. It exists only for public feedback and issue tracking.

## Before opening an issue

Please check existing issues first. If someone already requested the same file type, conversion, or feature, add a comment there instead of opening a duplicate.

## Request a new file type or conversion

When asking for new file support, include:

- The input file type you have
- The output file type you want
- Whether the file contains video, audio, images, text, metadata, subtitles, or attachments
- Whether you need this in the web app, desktop app, or both
- A sample file, if you can share one publicly

Do not upload private, sensitive, copyrighted, or personal files.

## Report a problem

For bugs, include:

- What you tried to convert
- What output you selected
- What happened
- What you expected to happen
- Your browser, operating system, and app version if relevant
